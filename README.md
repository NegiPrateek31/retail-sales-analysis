# Retail Sales Operations & Performance Analysis

## Project Overview
End-to-end retail analytics project focusing on data cleaning, validation, and business performance analysis using Python and Power BI.

## Dataset
Retail transaction data containing pricing, quantity, discounts, payment methods, categories, and transaction dates.

## Tools Used
Python (Pandas, NumPy), Power BI

## Key Work
- Cleaned and validated raw transaction data using Python
- Applied business rules to correct pricing and quantity inconsistencies
- Removed duplicate and invalid transactions
- Built an executive Power BI dashboard to analyze sales, quantity, discounts, and channel performance

## Key Insights
- Channel-wise performance (online vs in-store)
- Discounted vs non-discounted sales impact
- Category and product-level performance
- Daily transaction trends

## How to Run
1. Clone the repository
2. Install dependencies: `pip install pandas numpy`
3. Run the script: `python scripts/retail_automation.py`
